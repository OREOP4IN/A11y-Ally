WCAG 3.3.2 Violation: Form inputs without labels

Testing Instructions

  1. Load the A11y Ally extension in your browser
  2. Open this page (demo.html)
  3. Click the extension icon to view statistics
  4. Check the browser console for detected issues
  5. Inspect elements to see the fixes applied
  6. Look for data-a11y-ally-fixed="true" attributes on fixed elements

With API Keys Configured

If you've configured Cloud Vision API keys, the extension will:

  • Generate intelligent alt text for images
  • Analyze image content using AI
  • Create more descriptive labels

Without API Keys

Without API configuration, the extension will still:

  • Add basic alt text using filenames or generic descriptions
  • Fix all other accessibility issues
  • Provide useful fallback fixes

Viewing Results

After the extension processes this page, you can:

  • Open Browser Console: See detailed logs of issues found and fixes applied
  • Click Extension Icon: View statistics in the popup
  • Inspect Elements: Look for data-a11y-ally-fixed="true" attributes
  • Test Accessibility: Use screen readers or other tools to verify improvements
